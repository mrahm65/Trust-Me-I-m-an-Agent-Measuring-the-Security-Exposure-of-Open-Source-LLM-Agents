## SECTION 1: ICLR 2025 Agent Implementations

### 1. ExpoComm - Scalable Multi-Agent Communication
- **Repository:** https://github.com/LXXXXR/ExpoComm
- **Paper:** ICLR 2025
- **University/Lab:** Unknown (China)
- **Number of Agents:** Multi-agent (scalable communication topology)
- **Description:** Exponential Topology-enabled Scalable Communication in Multi-agent Reinforcement Learning. PyTorch implementation for MARL with IMP-MARL, MAgent environments.

### 2. Agent-Attack - Adversarial Robustness of Multimodal LM Agents
- **Repository:** https://github.com/ChenWu98/agent-attack
- **Paper:** ICLR 2025 (also Oral at NeurIPS 2024 Open-World Agents Workshop)
- **University/Lab:** Carnegie Mellon University
- **Number of Agents:** Single agent evaluation
- **Description:** Dissecting adversarial robustness of multimodal language model agents. Evaluates episode-wise and stepwise attacks on visual web agents.

### 3. AgentTrek - Agent Trajectory Synthesis
- **Repository:** https://github.com/xlang-ai/AgentTrek
- **Paper:** ICLR 2025 Spotlight
- **University/Lab:** Unknown
- **Number of Agents:** Browser-use agent
- **Description:** Agent Trajectory Synthesis via Guiding Replay with Web Tutorials. Cost-efficient pipeline for synthesizing high-quality agent trajectories. Achieves competitive performance on WebArena and MiniWob++.

### 4. Agent S - Open Agentic Framework
- **Repository:** https://github.com/simular-ai/Agent-S
- **Paper:** ICLR 2025
- **University/Lab:** Simular Research
- **Number of Agents:** Single GUI agent
- **Description:** An open agentic framework that uses computers like a human. Capable of performing general tasks across operating systems with multimodal grounding.

### 5. ACC-Collab - Multi-Agent LLM Collaboration
- **Repository:** https://github.com/LlenRotse/ACC-Collab
- **Paper:** ICLR 2025
- **University/Lab:** Unknown
- **Number of Agents:** Multi-agent (actor-critic collaboration)
- **Description:** An Actor-Critic Approach to Multi-Agent LLM Collaboration for improved decision-making.

### 6. AgentOccam - Simple Baseline for LLM-Based Web Agents
- **Repository:** https://github.com/amazon-science/AgentOccam
- **Paper:** ICLR 2025
- **University/Lab:** Amazon Science
- **Number of Agents:** Single web agent
- **Description:** A simple yet strong baseline for LLM-based web agents showing that simpler approaches can outperform complex agent designs.

### 7. Agent Security Bench (ASB)
- **Repository:** https://github.com/agiresearch/ASB
- **Paper:** ICLR 2025
- **University/Lab:** Zhejiang University / Rutgers University
- **Number of Agents:** 10 agents across 10 scenarios
- **Description:** Formalizing and benchmarking attacks and defenses in LLM-based agents. Covers 400+ tools, 27 attack/defense types, 10 scenarios including e-commerce, autonomous driving, finance.

### 8. CycleQD - Agent Skill Acquisition
- **Repository:** https://github.com/SakanaAI/CycleQD
- **Paper:** ICLR 2025
- **University/Lab:** Sakana AI, Japan
- **Number of Agents:** LLM skill acquisition framework
- **Description:** Agent Skill Acquisition for Large Language Models via Quality Diversity framework. Alternates task performance metrics as quality measures. Surpasses fine-tuning on coding, OS, and database tasks.

### 9. ScienceAgentBench - Scientific Discovery Agent Benchmark
- **Repository:** https://github.com/OSU-NLP-Group/ScienceAgentBench
- **Paper:** ICLR 2025
- **University/Lab:** Ohio State University (OSU NLP Group)
- **Number of Agents:** Benchmark for evaluating agents
- **Description:** 102 tasks from 44 peer-reviewed publications in 4 disciplines (Bioinformatics, Computational Chemistry, GIS, Psychology) for evaluating language agents on data-driven scientific discovery.

### 10. ADAS - Automated Design of Agentic Systems
- **Repository:** https://github.com/ShengranHu/ADAS
- **Paper:** ICLR 2025
- **University/Lab:** University of British Columbia / Vector Institute
- **Number of Agents:** Meta-agent that designs other agents
- **Description:** A meta agent that automatically programs ever-better agentic systems. Discovers novel agent designs that outperform hand-designed baselines by 13.6% on reading comprehension and 14.4% on math.

### 11. GAMABench - Multi-Agent Gaming Benchmark
- **Repository:** https://github.com/CUHK-ARISE/GAMABench
- **Paper:** ICLR 2025
- **University/Lab:** Chinese University of Hong Kong (CUHK-ARISE)
- **Number of Agents:** Multi-agent (2+ players)
- **Description:** Framework for evaluating LLMs' Gaming Ability in Multi-Agent environments. 8 classical game theory scenarios with dynamic scoring. Evaluated 13 LLMs from 6 families.

### 12. COMBO - Compositional World Models for Embodied Multi-Agent Cooperation
- **Repository:** https://github.com/UMass-Foundation-Model/COMBO
- **Paper:** ICLR 2025
- **University/Lab:** UMass Amherst / Tsinghua / Peking / MIT
- **Number of Agents:** 2-4 embodied agents
- **Description:** Learns compositional world models for multi-agent cooperation by factorizing joint actions and compositionally generating video for online cooperative planning.

### 13. Kinetix - General Agents Through Physics-Based Control
- **Repository:** https://kinetix-env.github.io
- **Paper:** ICLR 2025
- **University/Lab:** DeepMind / Google
- **Number of Agents:** Single agent
- **Description:** Investigating the training of general agents through open-ended physics-based control tasks in JAX.

### 14. Instant Policy - In-Context Imitation Learning
- **Repository:** https://www.robot-learning.uk/instant-policy
- **Paper:** ICLR 2025
- **University/Lab:** University College London (UCL)
- **Number of Agents:** Robot agent
- **Description:** In-context imitation learning via graph diffusion for robotic manipulation tasks.

### 15. MRQ - Model-Free Reinforcement Learning
- **Repository:** https://github.com/facebookresearch/MRQ
- **Paper:** ICLR 2025
- **University/Lab:** Meta FAIR
- **Number of Agents:** Single agent
- **Description:** Towards General-Purpose Model-Free Reinforcement Learning. Accelerating goal-conditioned RL algorithms and research.
