# Plan: Research 1,000 Individually Coded AI Agents

## Objective
Compile a comprehensive list of 1,000+ individually coded (working, runnable) AI agents from GitHub and other open-source repositories — not use cases, not ideas, but actual code implementations.

## Strategy
Multi-agent parallel research across different categories of agent sources:

### Stage 1 — Parallel Research (6 tracks)
1. **Track A**: Large curated collections (500-AI-Agents, awesome-ai-agents, etc.) — scan for repos claiming 100+ agents
2. **Track B**: Framework-specific agent galleries (AutoGPT, LangChain, CrewAI, AutoGen, LlamaIndex, etc.)
3. **Track C**: Domain-specific agent repos (coding, research, business, personal, gaming, etc.)
4. **Track D**: GitHub search — "AI agent" with code, filtering for repos with runnable implementations
5. **Track E**: Multi-agent systems and swarms (each system may contain 10-100+ individual agents)
6. **Track F**: Lesser-known individual agent repos (100+ individual repos each with unique agents)

### Stage 2 — Aggregation & Deduplication
Combine all findings, deduplicate, verify counts.

### Stage 3 — Output
Produce final structured list with repo name, agent count, source URL, and description.

## Output
- Markdown file: `/mnt/agents/output/1000-ai-agents-list.md`
